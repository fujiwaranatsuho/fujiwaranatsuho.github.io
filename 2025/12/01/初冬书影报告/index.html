啊哈 又有两个月不写书影报告了

《初吻》

远程获取内容失败，请检查 API 有效性。

属于是加在“想看”里但犹犹豫豫的电影，一是现在的坂元裕二+爱情片感觉很危险，感觉他一不小心就玩崩了，或者像花束的结构那样逐渐平庸。二是片源出来后几天，尝试看了十分钟的开头就没啥兴趣看下去了（而且松村北斗这个名字很赶客啊，和松隆子搭在一起更是奇怪）。这次捡回来纯粹是因为豆瓣评价高达8.5且感觉自己很久没有看过电影了觉得不行了，一定要复健文艺。于是赶着10月的尾巴把这一部给补完了。事实证明看完铺垫的部分，进入后面的穿越主线之后，看得非常顺畅（毕竟是坂元裕二的脚本）。
但说情节的话，我还是觉得很老套，这种穿越到过去改变命运的设定也不新鲜了。而且故事转折点也不是很能站得住脚——20年后的妻子向20年前的丈夫坦白一切，说两个人结婚之后很糟糕，一场谈话就能扭转十几年的婚姻，看到有短评说：婚姻就是男的改了就好了。虽然过于理想，但我没有特别反感这一点，我觉得电影对爱情的底色是悲观的：再相爱的人结婚之后都会被柴米油盐击碎——除非有重来的机会。如果不是女生能够穿越到过去，她和男主就只是一对室友——连怨侣都算不上，两个人已经没有任何多余的感情了。男女主是幸运地拥有了一次重写的机会，那没有机会的普通人，无论多么相爱，一周目还是会通往be结局。所以我觉得，这部电影看上去很感人，实际上对爱情的态度很冷漠（当然很有可能半圆只是觉得这么设定能推动情节，我过度解读了）。而且男主从什么时候开始变得不一样的呢？是变成社畜哈哈哈哈哈哈哈，这一点和《花束》真是一脉相承呢，看得出来半圆很讨厌工作了。这也是我比较喜欢这部电影的一个原因。
另外一个原因是，这部电影真的拍出了纯爱感。天呐！我多久没有看到好看的日本纯爱片了！一开始看卡司完全不抱希望，虽然比较好感松隆子，但她距离纯爱的年纪大概有二十年吧，现在也演不出《四月物语》了。松村北斗就更不用说了，在《黎明的一切》里面简直像一个木头，看到他的脸就烦。但《初吻》里面还不错，把男生悸动时的紧张、失措诠释得不错，至少是“生动”的。所以我居然，可以磕到！天呐（双手抱头）！但我可不会承认这是松村北斗的功劳，还是导演太会拍了，半圆太会写了。他如果再拿《黎明的一切》那种很大程度看演员发挥的文艺片，我看表现和之前也是半斤八两。
爱情喜剧果然是半圆的拿手好戏，中间女主追年轻男主那一段太有趣了，特别是女主为了听男主的告白，穿越了三次、换装了三次，满足自己的
有好几个场景能感到导演特意设计过，有点像包饺子的醋——比如用不同的丝线和卡片标注重大时间点以及故事的走向。真的挺像打完一周目之后二、三周目想打出其他的结局。

劝退我看《初吻》的原因之一是刷到了竹棘在上影节上映《初吻》之后的评价，怕剧透没有看具体的内容，但能感受到她不是很满意，所以我也降低了期待，没有那么大的热情打开这部电影。当我看完了电影，再去寻找她的觉得观感不好的地方——总觉得在钻牛角尖一般。她觉得故事的结尾很奇怪，两个人幸福地度过了十几年，丈夫依旧车祸离世，妻子不是更加难过了吗？女主不是第一世更悲惨吗？
我觉得这个观点对我不太成立。过程比结局更加重要，虚度十几年的二人时光难道就好过两人冷漠共处最后对“陌生人”离世毫无波澜吗？这又引发我对于评价作品的思考，观众或读者究竟该不该评价故事的走向呢？因为这个故事走向不符合我的期待，便给差评？至少在网文的领域，我的想法是读者接受作者的表达而非指手画脚。be和he没有优劣之分，但是情节之间的逻辑是否顺畅、情节是否有意义、角色塑造是否有问题，这些都可以批评。
但在电影领域里面，好像也听到过《反派影评》经常批评电影的情节有问题（太久没听了举不出具体的例子），所以我只能困惑。目前的想法就是，如果作者能自圆其说、逻辑自洽就没有问题。
21年花束，23年怪物，25年初吻，豆瓣评分都在8.5之上，真正的日影紫微星——坂元裕二！
但，请不要写什么初恋、初吻之类的好吗？过于老套，我不想看纯爱了，谢谢。（实在要写纯爱就写《怪物》那种吧，爱看）

《关于偶像，一边纠结一边思考》

远程获取内容失败，请检查 API 有效性。

刷到健身搭子的阅读广播（她阅读量远超我），看到标题心想这不是我的领域吗？便想找来读尝尝咸淡看看水平高低。
果不其然我很失望哈哈哈。
首先，这是一本论文（能称得上是论文吗？）集，不同的作者自选主题，这导致了整本书的研究领域很模糊。我觉得男性偶像和女性偶像的情况大相径庭，不能混为一谈。每次看到文中说“粉丝对偶像的情感……”，我的脑子就充满了问号，你到底是想说男性偶像还是女性偶像。
其次，由于自选主题，每篇文章的篇幅都不是很长，对一个论题都是浅尝辄止，我甚至都没有看出这篇文章究竟想表达什么。
然后，因为论文的研究不深入，不同的读者群体的阅读收获都不多。对于像我这样的偶像宅，我觉得里面大部分说的内容我都知道，甚至思考得更加深入。而对于不追星的读者，文章科普的部分不多，难以理解追星的现状以及粉丝的情感。
最后，完全看不出这本书里面的研究方法，感觉完全是在自说自话。“粉丝对偶像总是……”等等！你是怎么得出这个结论的，有调查过吗？怎么就当成一个定论普通地叙述出来了？是不是我闭门造车，一拍脑袋也能写出一篇论文出来。日本的社科界还是太好混了啊，看得我心动。
整本书读下来，我最related的部分居然是——对早安少女和hp系的科普。是的！她们很好！希望她们大火！
原本想夹带私货地介绍一下我追的女团来着，但肯定又是长篇大论，所以在此报菜名好了。追的有符号系（夸她们的话可以单独开一篇博客了）和开闭，关注的有hp系的早安（juice=juice和山茶花工厂也会看一下，过世的玉兰花工厂我很喜欢），星辰系的私立惠比寿中学（但是这团老成员差不多都走完了，前两年出的歌都很难听）和超心动宣传部，还会听许多乐曲派的歌，在此不一一列出，博客左下角音乐放的就是我的乐曲派歌单。
